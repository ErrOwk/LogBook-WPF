## eLogbook of WPF
> 一个简单基础的电子通联日志

该项目旨在让用户可以简单快速的记录QSO并且便于导出/同步到LoTW和QRZ.com等平台.

### 功能
- 添加/编辑/删除 QSO
- 自动填充字段
- 导出adi
- *同步到LoTW和QRZ.com* (写出来了但是未验证可用~~因为我也还没有呼号只能靠别人测试~~)
